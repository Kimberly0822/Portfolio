Notes

This project I had to make a web ad. I struggled on what exactly to do then remembered the old ads video game companies had that would rotate through game covers to show what they're selling and entice you to buy it. So figured why not make one for Playstation Plus

  • Fluid Brush Tool, Gradient Tool, and Object Drawing Mode
    • With the help of these I was able to make the Playstation Plus logo. I drew out the symbol then went into object drawing mode to make sure the lines looked how I wanted. Then with the help og the gradient tool I made a custom color to give that kinda shine and shadowed look to it
  • Motion Tween
    • The motion tween let me create the spinning effect of the video game covers. This took me quite some time to do because the way layers work once the front games reached the back they'd still be layering over top the new front covers. In order to fix this I had to create two different layers for each cover. One for when it was in the front and another for when it was in the back and and then have them line up perfectly once the layer switched so it'd look seamless
