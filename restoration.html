For this project I attempted to restore an old photograph and I used many tools to help me with this process such as...

  • The Healing Brush Tools
    • These allowed me to remove any damage from the picture
  • The Patch and Lasso tools
    • I used these to select the weird spot in the trees and then the patch tool to cover it up with a non damaged section
  • Neural Filters
    • I selected the colorize filter with these which is why the picture is now colored
  • Sky Replacment tool
    • With this tool I was able to replace the sky. I picked one that had a cloudy look to it since there wasn't much harsh lighting on the soldier. Plus the colors looked pretty :D
